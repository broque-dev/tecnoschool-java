# TecnoSchool – Sistema de Gestión Académica

TecnoSchool es una aplicación de escritorio desarrollada en Java que permite la gestión académica de una institución educativa. El sistema facilita la administración de estudiantes, matrículas, asistencia y calificaciones, integrando funcionalidades clave para el control académico.

Este proyecto fue desarrollado como parte de mi formación en Ingeniería de Sistemas, aplicando conceptos de programación orientada a objetos y manejo de bases de datos.

## Tecnologías utilizadas

* Java
* MySQL
* NetBeans
* Programación Orientada a Objetos (POO)

## Funcionalidades principales

* Registro y gestión de estudiantes
* Matrícula académica
* Control de asistencia
* Registro de calificaciones
* Gestión de cursos
* Interfaz de escritorio para administración

## Cómo ejecutar el proyecto

1. Clonar el repositorio
2. Abrir el proyecto en NetBeans
3. Configurar la conexión a la base de datos
4. Ejecutar el proyecto desde el IDE
